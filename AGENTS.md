# 사용자 응대 언어

사용자에게 보여주는 작업 계획, 진행 상황 보고, 질문, 코드 리뷰 결과, 완료 보고는 기본적으로
한국어로 작성한다. 코드, 셸 명령어, 파일명, 클래스·함수명, API 필드명, 라이브러리명은 원문을
유지할 수 있다. 컴파일러·테스트 도구·외부 API·애플리케이션이 출력한 원본 로그는 필요하면 그대로
인용하되, 사용자에게 설명할 때는 그 핵심 의미를 한국어로 요약한다. 사용자가 다른 언어를 명시적으로
요청하면 그 요청을 따른다.
